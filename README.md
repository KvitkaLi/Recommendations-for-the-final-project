# Recommendations-for-the-final-project
Рекомендації до випускної роботи для студентів ІТ школи ПДМ
# Формат виконання роботи
## Випускна робота виконується у вигляді:  
•	односторінкового сайту (landing page) або  
•	невеликого сайту з 2–3 сторінок.  
Верстка виконується виключно за готовим макетом у Figma, наданим викладачем або погодженим із ним.
________________________________________
# Технічні вимоги до роботи  
## Обов’язково:  
•	HTML5 
* Використання семантичних тегів -- header, nav, main, section, footer    
* Структурований та логічний контент  
* Окремий CSS-файл  
* Flexbox або Grid  
* Адаптивність (мінімум 1 медіазапит)  
* Єдиний стиль оформлення  
## Додатково (за бажанням):  
•	Hover-ефекти  
•	Простi анімації  
•	Google Fonts  
•	Іконки (SVG / font icons)  
________________________________________
# Назва проєкту
# Рекомендації до вибору проекту(назви проекту)
1. Особисті та презентаційні сайти
(добре мотивують, легко перевіряти індивідуальність)  
•	Персональний сайт учня («Про мене»)  
•	Моє хобі / захоплення (спорт, музика, малювання, програмування)  
•	Портфоліо учня (навчальні або творчі роботи)  
•	Моя майбутня професія  
•	Сайт про улюблену книгу / фільм / серіал  
________________________________________
2. Освітні та інформаційні проєкти
(акцент на структурі та семантиці)  
•	Сайт шкільного предмета (інформатика, історія, біологія)  
•	Мініенциклопедія на одну тему (тварини, планети, винаходи)  
•	Навчальний сайт «Основи HTML/CSS для початківців»  
•	Сайт-пояснення теми з курсу (наприклад, «Що таке Інтернет»)  
•	Онлайн-конспект або довідник  
________________________________________
3. Соціальні та громадські теми  
(формують змістовну роботу з текстом)  
•	Безпека в Інтернеті для підлітків  
•	Екологія та збереження природи  
•	Волонтерство та добрі справи  
•	Здоровий спосіб життя  
•	Медіаграмотність для школярів  
________________________________________
4. Тематичні сайти та події  
(добре підходять для landing page)  
•	Сайт шкільного заходу або гуртка  
•	Сайт табору / туристичної мандрівки  
•	Сайт фестивалю або конкурсу  
•	Афіша подій міста / школи  
•	Календар важливих дат
